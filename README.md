# 🏃 Run Fight Game

A modern HTML5 Hyper Casual Runner Game inspired by popular mobile games.

---

## 🎮 Features

- 🏃 Endless Runner
- 🍔 Food Collection
- 🏋️ Dumbbell Collection
- ⚔️ Enemy Fight
- 👹 Boss Battle
- 🎁 Reward Chest
- 💰 Coin System
- 📈 Level Progress
- 🎨 Colorful 3D UI
- 📱 Mobile Responsive
- 💻 HTML + CSS + JavaScript
